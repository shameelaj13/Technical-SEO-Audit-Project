# Technical SEO Audit Project

## Project Overview
This is a beginner Technical SEO Audit project for a WordPress website.
The project includes website analysis, technical issues identification, and SEO recommendations.

## Audit Areas Covered
- Website Structure Analysis
- Robots.txt Review
- XML Sitemap Analysis
- Page Speed Optimization
- Mobile Friendliness
- Schema Markup Implementation
- On-Page SEO Checks

## Tools Used
- Google PageSpeed Insights
- Google Search Console
- Google Rich Results Test
- WordPress (Yoast SEO / Rank Math)

## Files Included
- Technical SEO Report (.docx)
- SEO Audit Excel Sheet (.xlsx)
- Presentation Slides (.pptx)

## Objective
- To analyze and improve website technical SEO performance for better search engine visibility.
